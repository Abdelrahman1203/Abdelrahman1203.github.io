# TRC — Real Estate Referral Platform

## One-line summary
Real estate referral and membership platform with admin web panel and mobile REST API for property developers, referrals, wallets, and commissions.

## Tech stack
- **Backend:** Laravel 10, PHP 8.1+, MySQL, Breeze, Sanctum, Spatie Permission
- **Frontend:** Blade admin + Tailwind CSS 3 + Alpine.js 3 + Vite 5
- **Export:** DomPDF, Maatwebsite Excel | **Push:** Google API Client (FCM)

## Key features
- **Admin:** Developers, projects, units, referral pipeline with commission tracking
- Provider onboarding with staged workflow and Excel/PDF exports
- User verification, wallet admin, withdrawal reports, membership types
- **Mobile API:** Property browsing, favorites, referrals, wallet/deals/stages, FCM notifications
- Account managers, complaints, home page CMS

## CV bullet points
- Developed **TRC**, a real estate referral platform (Laravel/MySQL) with admin referral pipeline, commission wallets, and property catalog API.
- Built multi-stage provider/referral workflows with Excel/PDF export and FCM push notifications.
