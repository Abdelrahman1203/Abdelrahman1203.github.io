# BlackHorse — B2B Wholesale & Delivery Platform

## One-line summary
Multi-sided B2B wholesale and delivery platform for supermarkets, suppliers, and delivery drivers with real-time orders, group purchasing, and admin ERP.

## Tech stack
- **Backend:** Node.js, Express.js, MongoDB, Mongoose, Socket.IO
- **Auth:** JWT, bcrypt, SMS OTP, phone verification
- **Real-time:** Socket.IO (orders, groups, chat)
- **Integrations:** Firebase push, Cloudinary, Nodemailer, Cron jobs
- **Admin UI:** Flutter Web | **Mobile:** Flutter (customer + sales agent)

## Key features
- Multi-role auth: admin, supplier, customer, delivery driver
- Order lifecycle, partial returns, delivery assignment, district analytics
- Group purchasing with automated expiry cron jobs
- Supplier catalog: products, offers, promo codes, balance sheets
- Admin ERP: inventory, treasury, purchases/sales, fines, reports
- Real-time chat and live order updates via Socket.IO

## CV bullet points

- Built 240+ REST API endpoints with Postman documentation, Firebase push notifications, and a Flutter Web admin panel.
