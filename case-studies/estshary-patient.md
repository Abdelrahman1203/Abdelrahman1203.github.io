# Case Study: Estshary Patient Website

## Overview

Public-facing companion to the [Estshary Admin](./estshary-admin.md) platform. Shares the same MySQL database for real-time doctor availability.

## Key Deliverables

- Homepage with sliders and advanced doctor search
- AJAX filtering by city, area, and specialization
- Doctor profile pages with available time slot selection
- Service and medical operation booking flows
- Enterprise/clinic profiles with filtered reviews

## Business Impact

- Reduced phone call volume for appointment inquiries
- 24/7 online booking availability for patients
- Shared database eliminated data sync issues between admin and public site

## Technologies

Laravel 10 · Blade · Bootstrap · jQuery · MySQL
