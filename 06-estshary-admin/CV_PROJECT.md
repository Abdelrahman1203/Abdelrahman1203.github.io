# Estshary — Healthcare Admin & API

## One-line summary
Full healthcare management admin panel and REST API for doctor/clinic booking platform with AI chatbot, Firebase notifications, and mobile APIs.

## Tech stack
- **Backend:** Laravel 10, PHP 8.1+, MySQL, JWT Auth, Sanctum
- **Permissions:** Spatie Permission + Activity Log
- **Frontend:** Blade, Bootstrap 5, Tocly admin template, Vite, Sass, RTLCSS
- **Integrations:** Firebase push, OpenAI/ChatGPT, Tesseract OCR, DomPDF, mPDF, BeonSMS

## Key features
- Doctor & clinic management: schedules, vacations, approvals, multi-branch enterprises
- Appointment/reservation system with prescriptions and reports
- Review moderation with multi-dimensional ratings
- Financial module: invoice generation per doctor/clinic, PDF download
- Ads system: city/area/Google/social ads with reporting
- AI chatbot API: symptom/location-based doctor matching via ChatGPT
- Doctor mobile API: OTP signup, clinic creation, reservation management
