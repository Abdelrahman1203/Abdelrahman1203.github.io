# ERP Arabia — Warehouse & Inventory ERP

## One-line summary
Vue.js 3 refactored ERP for multi-branch warehouse operations: purchases, transfers, stock adjustments, and rich reporting with Arabic RTL support.

## Tech stack
- **Backend:** Laravel 10, PHP 8.1+, MongoDB, Redis, Sanctum
- **Frontend:** Vue 3, Pinia, Vue Router, Vue I18n, Tailwind CSS, Vite
- **Export:** pdfmake, jsPDF, mPDF, PhpSpreadsheet, SendGrid
- **Arabic:** arabic-persian-reshaper, bidi-js, full RTL document support

## Key features
- Settings: countries, companies, branches, users, roles, suppliers, units, tax files
- Warehouse: categories, stock levels, bulk entry, inter-warehouse transfers
- Items: hierarchical categories, images, alternative units, bulk price entry
- Purchases: approval workflow, grouped invoices, purchasing orders report
- 10+ reports with PDF/Excel export; dashboard analytics
- Bilingual EN/AR admin SPA with role-based permissions
