# Harvest British College (SYS) — Education ERP & CRM

## One-line summary
Modular Laravel platform for Harvest British College: CRM lead pipeline, class/group management, payments, multi-portal access (admin, instructor, student), and bilingual public website — live at harvestcollege.co.uk.

## Tech stack
- **Backend:** Laravel 10, PHP 8.1+, MySQL, nwidart/laravel-modules
- **Modules:** Website, CustomerPortal, InstructorPortal
- **Packages:** Livewire 3.5, Spatie Permission & Activity Log, Maatwebsite Excel, Twilio, JWT, Pusher, Socialite, Sanctum, Full Calendar
- **Frontend:** Blade, Livewire, Laravel UI, bilingual EN/AR

## Key features
- **Admin ERP:** Lead CRM, follow-ups, groups/classes, schedules, payments, safes, HR (employees, vacations, recruitment)
- **Operations:** Placement tests, exam applicants, certificates, warehouse items, event calendar
- **Reports:** Sales, follow-up, attendance, activation, filling ratio, client/customer trackers (20+ reports)
- **Customer portal:** Registration, OTP login, course enrollment, payments, placement tests
- **Instructor portal:** Classes, attendance, exams, makeup sessions, monitoring, ratio reporting
- **Public website:** CMS-driven pages, offers, events, gallery, branch locator, job applications
